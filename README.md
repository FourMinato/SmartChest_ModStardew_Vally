# 📦 SmartChest - Stardew Valley Mod

มอดจัดการหีบเก็บของอัจฉริยะที่ช่วยให้ฟาร์มของคุณเป็นระเบียบด้วยระบบหมวดหมู่ที่ปรับแต่งได้เอง (Centering & Polished UI).

## ✨ คุณสมบัติเด่น (Features)
- **Custom Categories:** กำหนดหมวดหมู่ไอคอนและสีให้กับหีบแต่ละใบ
- **Smooth GUI:** เมนูจัดการหมวดหมู่ที่สวยงาม ใช้งานง่าย และรองรับการย่อขยายหน้าจอ (Responsive)
- **Pixel Perfect:** ปรับจูนขนาดไอคอนให้กึ่งกลางช่องพอดีเป๊ะเพื่อความสวยงามระดับพรีเมียม
- **Localization:** รองรับการแปลภาษา (ปัจจุบันมีภาษาไทยและอังกฤษ)

## 🛠️ ความต้องการ (Requirements)
- [SMAPI - Stardew Modding API](https://smapi.io/)
- [GenericModConfigMenu](https://www.nexusmods.com/stardewvalley/mods/5098)
- Stardew Valley เวอร์ชั่น 1.6 ขึ้นไป

## 🚀 วิธีการติดตั้ง (Installation)
1. ติดตั้ง SMAPI เวอร์ชั่นล่าสุด
2. แตกไฟล์มอดลงในโฟลเดอร์ `Mods` ในที่ที่คุณติดตั้งเกม
3. เข้าเกมผ่าน SMAPI และเริ่มจัดระเบียบหีบได้เลย!

## 🤝 การใช้งานร่วมกับมอดอื่น (Compatibility)
- ทำงานร่วมกับมอด **QuantumStorage** ได้อย่างสมบูรณ์แบบ (จะแสดงไอคอนหมวดหมู่บนหน้าต่าง QuantumStorage โดยอัตโนมัติ)
